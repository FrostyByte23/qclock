
## qclock (quick-clock)

`qclock` is a tiny CLI written in C# with the [Spectre.Console library](https://spectreconsole.net/). It is customizable with its `config.json` file, where you can edit the shape of the panel, the colors, and the time format.

`qclock` is designed for **Linux**. It may work on Windows, however, I am only testing it on Linux.
